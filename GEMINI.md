# Project Engineering Standards

## SQL Output Convention
- **Complete Statements Only**: When providing SQL, always output the **full, complete SQL statement**.
- **Idempotency**: Ensure SQL statements are idempotent (e.g., using `IF NOT EXISTS`, `REPLACE INTO`, etc.) where appropriate, following `references/sql-idempotency.md`.
- **No Partial Snippets**: Strictly forbidden to provide only partial or fragmented SQL code.

## Language & Documentation
- **Chinese Response**: All technical explanations, comments, and interactions must be in **Chinese** as per global preferences.
